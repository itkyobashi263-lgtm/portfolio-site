# 自己紹介・ポートフォリオサイト

私の自己紹介および、これまでの学習成果をまとめたポートフォリオサイトです。

## 🌐 Live Demo
[Live Demo URL (GitHub Pages)](https://itkyobashi263-lgtm.github.io/portfolio-site/)

## 🛠 掲載している制作実績の主な使用技術
- **ポートフォリオサイト本体**: HTML5, CSS3, Vanilla JavaScript (セマンティックなマークアップ、レスポンシブデザイン)
- **事業所内ポータルサイト**: React (TypeScript), PHP
- **AIチャットボット (SK-Nexus)**: React, Python (FastAPI), Docker, OCI (Oracle Cloud)
- **タスク割り当てアプリ (Task Manager)**: PHP, MySQL
- **オリジナルテーマ制作**: WordPress

## 💡 ポートフォリオサイト本体の実装の特徴
- **フルスクラッチでのスタイリング**: フレームワークに頼らず、カスタムCSSを用いてレイアウトを構築しました。
- **UIの実装**: Vanilla JSを用いて、ハンバーガーメニューの開閉制御や、スクロールに連動した動きのあるUI、プロジェクトタブ切り替えを実装しました。
- **セマンティックなマークアップ**: 適切なHTMLタグを使用し、文書構造の明確化とアクセシビリティを意識しています。
