# 自己紹介・ポートフォリオサイト

私の自己紹介および、これまでの学習成果をまとめたポートフォリオサイトです。

## 🌐 Live Demo
[Live Demo URL (GitHub Pages)](https://itkyobashi263-lgtm.github.io/portfolio-site/)

## 🛠 使用技術
- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript

## 💡 実装の事実と特徴
- **フルスクラッチでのスタイリング**: フレームワークに頼らず、約680行のカスタムCSSを用いてレイアウトを構築しました。
- **レスポンシブ対応**: メディアクエリを活用し、スマートフォン・タブレット・PCの各ブレイクポイントに最適化したUIを実装しています。
- **UIの実装**: Vanilla JS（約110行）を用いて、ハンバーガーメニューの開閉制御や、スクロールに連動した動きのあるUIを実装しました。
- **セマンティックなマークアップ**: 適切なHTMLタグを使用し、文書構造の明確化とアクセシビリティを意識しています。
